### Mediazone
Self hosted songs hosting service. Upload, Download and Play your favorite songs

### Prequestics
```.env
Python 3
```

## How to install and run the app
```.env
cd application-path
pip3 install pipenv
./run.sh
```

### Build and run via docker
```.env
 docker build -t songs:VERSION .
 docker run -p 5000:5000 songs:VERSION
```

### Open below url
```.env
http://localhost:5000
```
